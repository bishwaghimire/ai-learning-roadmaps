## Prerequisites

Before starting Data Science, you should be comfortable with:

- **Mathematics**: Basic Algebra, Statistics, Probability  
- **Programming Language**: Python  
- **Tools & Skills**:  
  - NumPy  
  - Pandas  
  - Matplotlib / Seaborn  
  - Jupyter Notebook  

---

## Beginner Level — Foundations of Data Science

Build a **strong foundation** in Python, data handling, and basic exploratory analysis.

### Core Concepts:
[Python Basics](https://docs.python.org/3/tutorial/),
